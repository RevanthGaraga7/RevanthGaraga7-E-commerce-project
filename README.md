Samanyastra – E-Commerce End-to-End Web Application

A complete E-Commerce Web Application designed to simulate a real online shopping platform.
This project demonstrates core frontend development skills, authentication flow, product catalog browsing, shopping cart functionality, and order management.

The goal of the project is to showcase how modern e-commerce platforms operate with features like user authentication, cart system, order tracking, and customer support modules.
